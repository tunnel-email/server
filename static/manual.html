Начало работы

Скачайте клиентское приложение с главной страницы. Доступны версии для Windows, Linux, MacOS.

Распакуйте zip-архив (ПКМ в проводнике → Извлечь)

Запустите исполняемый файл

Нажмите "Начать"

Стартовый экран приложения

Авторизация

После нажатия кнопки "Начать", войдите в аккаунт Яндекс в открывшемся окне в браузере

Нажмите "Я завершил авторизацию" или "Повторить", если произошла ошибка

Окно авторизации

Если вы столкнулись с проблемами входа, убедитесь, что у вас установлен браузер на компьютере

Использование временной почты

После успешной авторизации, вы можете создавать временные почтовые ящики. Для этого нажмите на "Создать новую почту"

Ожидайте получения TLS-сертификата

Процесс получения TLS-сертификата

После того, как показался интерфейс почтового клиента, всё готово для получения электронных писем. На изображении обозначено, для чего нужен каждый элемент интерфейса

Интерфейс почтового клиента

Чтобы получить письмо, просто нажмите на кнопку "Копировать" и используйте ваш адрес при регистрации на любом сервисе.

Для получения сертификатов по умолчанию используется Let's Encrypt, однако есть и возможность использовать ZeroSSL в качестве альтернативы

Проверка безопасности

Чтобы проверить, что на вашу почту только вы имеете сертификат, перейдите в дополнительные функции в почтовом клиенте (доступно по кнопке ⋮) и нажмите "Проверить безопасность". Проверка безопасности доступна не сразу, обычно нужно подождать несколько минут после создания почты

Раздел проверки безопасности

Использование ZeroSSL

Вы можете использовать ZeroSSL вместе Let's Encrypt для получения сертификатов. Это может быть полезно, если Let's Encrypt временно недоступен.

Обратите внимание, что на данный момент ZeroSSL не работает в России

Выберите ZeroSSL, перед тем как создать почту

Выбор провайдера SSL-сертификатов

Нажмите "Открыть сайт ZeroSSL"

Войдите или создайте аккаунт ZeroSSL, перейдите во вкладку Developer и скопируйте Developer token

Введите токен и нажмите "Сохранить"
